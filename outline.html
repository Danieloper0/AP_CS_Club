Club Outline

Below are the club outline of the year. (Note that it can change)

Week Topic Content Outline Resources AP CS A or AP CS P?
Intro week
1 Introduction to Programming & Computing Systems Overview of programming (Java and Python), setting up environments, basic syntax, and an introduction to how computers and computing systems work. Codecademy Java, Python.org tutorial, Code.org AP CS A & AP CS P
2 Variables and Data Types Understanding different data types, variables, constants, and basic operations on them. W3Schools, Java documentation AP CS A
3 Control Structures Conditionals (if statements), loops (for, while), and switch statements. Java tutorials on control structures AP CS A
4 Functions/Methods Defining and calling functions, parameters, return types, and scope. Codecademy Java Functions AP CS A
5 Data Structures Arrays, lists, and maps. Java documentation, W3Schools AP CS A
6 Object-Oriented Programming Classes, objects, inheritance, and polymorphism. Codecademy Java OOP AP CS A
7 AP Exam Preparation Review of topics covered, practice exams, and exam strategies. AP Classroom, College Board resources AP CS A
8 Introduction to Algorithms Understanding basic algorithms, complexity analysis, and problem-solving strategies. GeeksforGeeks, Khan Academy AP CS A
9 Sorting Algorithms Study of various sorting algorithms (e.g., bubble sort, quick sort, merge sort) and their complexities. Visualgo.net, Codecademy AP CS A
10 Searching Algorithms Linear and binary search algorithms. GeeksforGeeks, Java documentation AP CS A
11 Recursion Understanding recursion, base cases, and recursive solutions. Khan Academy, W3Schools AP CS A
12 More Data Structures Introduction to stacks, queues, and linked lists. Java documentation, W3Schools AP CS A
13 AP CS A Topics Review Review and discussion of key AP CS A topics and concepts. AP Classroom, College Board resources AP CS A
14 Introduction to Web Development Basics of HTML, CSS, and JavaScript. W3Schools, Codecademy AP CS P
15 Web Development Project Work on a simple web project applying HTML, CSS, and JavaScript skills. MDN Web Docs AP CS P
16 Data Collection and Analysis Understanding data collection methods and analysis techniques. W3Schools, Google Data Analytics AP CS P
17 AP CS P Concepts Review Review and discussion of key AP CS Principles topics and concepts. AP Classroom, College Board resources AP CS P
18 Introduction to Cybersecurity Basic concepts of cybersecurity and data protection. Cyber.org, Khan Academy AP CS P
19 Ethics in Computing Understanding ethical implications in technology and computing. Code.org, Khan Academy AP CS P
20 Final Project Brainstorming Start brainstorming and planning for final projects. AP Classroom AP CS A & P
21 Project Work Session Work on final projects in groups. Class resources, mentors AP CS A & P
22 Project Work Session Continue working on final projects. Class resources, mentors AP CS A & P
23 Project Presentations Present final projects to the class. Class resources AP CS A & P
24 Project Feedback Provide and discuss feedback on projects. Class resources AP CS A & P
25 Review for Final Exam Review major topics and concepts for the final exam. AP Classroom, College Board resources AP CS A & P
26 Practice Exam Take a practice exam to prepare for the final. AP Classroom AP CS A & P
27 Final Exam Review Final questions and review before the exam. AP Classroom, College Board resources AP CS A & P
